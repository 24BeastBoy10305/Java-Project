##  How to Run
1. Clone the repository:
   ```bash
   git clone https://github.com/24BeastBoy10305/Java-Project
   ```

2. Enter the project folder:
   ```bash
   cd Java-Project/java_project/src
   ```

3. Compile and run (replace `Main` with the actual main class file):
   ```bash
   javac Main.java
   java Main
   ```
